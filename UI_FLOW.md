## 📊 Частоты рассылки

### Опция 1: Каждый час
- **Когда:** Каждый час в :00
- **Для кого:** Активные пользователи, готовятся к собесу через неделю
- **Плюсы:** Максимальная тренировка
- **Минусы:** Может надоесть

### Опция 2: Раз в день (9:00)
- **Когда:** Каждый день в 9:00 MSK
- **Для кого:** Умеренная подготовка, есть время
- **Плюсы:** Не надоедает, стабильно
- **Минусы:** Мало практики

### Опция 3: 2 раза в день (9:00, 18:00)
- **Когда:** Утром и вечером
- **Для кого:** Сбалансированная подготовка
- **Плюсы:** Не перегружает, хорошая практика
- **Минусы:** —

### Опция 4: 3 раза в день (9:00, 14:00, 20:00)
- **Когда:** Утро, обед, вечер
- **Для кого:** Интенсивная подготовка
- **Плюсы:** Много практики
- **Минусы:** Может быть слишком много

## 🛠️ Техническая реализация

### БД (users таблица)
```sql
ALTER TABLE users ADD COLUMN auto_training_enabled BOOLEAN DEFAULT FALSE;
ALTER TABLE users ADD COLUMN auto_training_frequency VARCHAR(20) DEFAULT 'hourly';
-- Варианты: 'hourly', 'daily', 'twice_daily', 'thrice_daily'
```

### Планировщик (cron)

**Для каждой частоты — свой trigger:**

```python
# Каждый час
CronTrigger(minute=0)

# Раз в день (9:00)
CronTrigger(hour=9, minute=0)

# 2 раза в день (9:00, 18:00)
CronTrigger(hour='9,18', minute=0)

# 3 раза в день (9:00, 14:00, 20:00)
CronTrigger(hour='9,14,20', minute=0)
```

### Callback Data Structure
```
Format: training_auto:action:param

Examples:
- training_auto:menu         → Открыть меню настройки
- training_auto:enable       → Включить (открыть выбор частоты)
- training_auto:set:hourly   → Установить "каждый час"
- training_auto:set:daily    → Установить "раз в день"
- training_auto:disable      → Отключить рассылку
- training_auto:back         → Назад
```

## 🧪 User Flow (тестирование)

**Сценарий 1: Включение рассылки**
1. `/learning` → [📬 Рассылка: ❌ Выкл]
2. Открылось меню → [✅ Включить]
3. Выбор частоты → [⏱️ Каждый час]
4. ✅ "Рассылка включена! Частота: Каждый час"
5. Проверить БД: `auto_training_enabled = TRUE`, `frequency = 'hourly'`

**Сценарий 2: Смена частоты**
1. `/learning` → [📬 Рассылка: ✅ Каждый час]
2. Открылось меню → [⚙️ Изменить]
3. Выбор частоты → [📅 Раз в день]
4. ✅ "Настройки обновлены! Частота: Раз в день (9:00)"

**Сценарий 3: Отключение**
1. `/learning` → [📬 Рассылка: ✅ Каждый час]
2. Открылось меню → [🛑 Отключить]
3. ✅ "Рассылка отключена"
4. Проверить БД: `auto_training_enabled = FALSE`
