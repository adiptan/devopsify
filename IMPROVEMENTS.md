**Решение:**

#### A. Добавить кнопку в режиме обучения

**В `/learning` добавить кнопку:**
```

```

**Логика:**
- При нажатии → объяснить как работает + подтвердить
- Добавить user_id в БД (`auto_training_enabled = True`)
- Добавить user_id в планировщик (`add_user_to_scheduler`)
- Отправить: "✅ Автоматическая рассылка включена! Ты будешь получать задачу каждый час."

#### B. Добавить команду `/training_auto`

**Команда:**
```
/training_auto
```

**Логика:**
1. Проверить статус (включена ли рассылка)
2. Если выключена → показать кнопки:
   ```
   📬 Автоматическая рассылка задач

   Получай задачу каждый час автоматически!

   [✅ Включить] [❌ Нет, спасибо]
   ```
3. Если включена → показать:
   ```
   ✅ Автоматическая рассылка включена
   Ты получаешь задачу каждый час.

   [🛑 Отключить]
   ```

**Добавить колонку в таблицу `users`:**
